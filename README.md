# Streaming Educativo

Aplicación de consola desarrollada en Python conectada a Microsoft SQL Server mediante pyodbc.

## Funcionalidades

- Visualización de tablas.
- Inserción de registros.
- Actualización de registros.
- Eliminación con validación de integridad referencial.
- Consultas SQL avanzadas.

## Tecnologías

- Python 3
- SQL Server 2022
- pyodbc
- Visual Studio Code
